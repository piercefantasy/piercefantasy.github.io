网站配置

注册github账号

  • 注册账号：piercefantasy

本地配置

Git下载安装

  • 下载：https://git-scm.com/downloads
  • 安装：双击安装包，一路点Next
  • 检验：鼠标右键会有 Git Bash Here，这是Git的命令行

Nodejs下载安装

  • 下载：https://nodejs.org/zh-cn/download
  • 安装：双击安装包，一路点Next
  • 检验：
    • node：打开cmd窗口，执行命令 node -v 查看node版本 v18.16.0
    • npm：在安装node的同时也安装了npm，执行 npm -v 查看npm版本 9.5.1

安装Hexo

  • 建立博客文件夹：Blog（右键点击GitBash，打开git的控制台窗口，以后所有操作都在git控制台进行）
  • 安装Hexo：
    • 安装：npm install hexo-cli -g
    • 检验：hexo -v
  • 初始化文件夹：hexo init
  • 安装必备组件：
    • npm install
    • npm i hexo-deployer-git
    • npm install hexo-deployer-git –save
    • npm i hexo-renderer-swig
  • 指令：hexo clean && hexo g && hexo s –debug
    • 清除缓存：hexo clean
    • 生成静态网页：hexo g
    • 打开本地服务器：hexo s –debug
  • 浏览器打开本地网页：http://localhost:4000/

连接网站Github与本地

github → Git

在命令行中输入如下命令，配置Git用户名和邮箱地址（连接Github与本地）

  • git config –global user.name “piercefantasy”
  • git config –global user.email “2580306492@qq.com“

Git→github

在命令行中输入如下命令，本地SSH keys保存到github

  • ssh-keygen -t rsa -C “2580306492@qq.com“
  • cat ~/.ssh/id_rsa.pub
  • 内容：ssh-rsa ···= 2580306492@qq.com
  • 打开网址https://github.com/settings/keys ，New SSH key，填入上段内容
  • 检验成功与否：ssh -T git@github.com

github→博客配置

  • github获取token：Settings→Developer settings→tokens (classic)
  • 配置博客_config.yml文件，这是博客的配置，修改最后一行的配置：
    deploy:
    type: git
    repo: git@github.com:piercefantasy/piercefantasy.github.io.git
    branch: main
    token: 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
